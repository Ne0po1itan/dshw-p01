### 数据来源
- 股票行情：[baostock]，后复权，日度
- 市场指数：[创业板指（399006），因其与沪深300风格差异显著、β值高，便于清晰验证CAPM中“高风险对应高收益”的核心逻辑。]
- 宏观指标：[M2同比增速]，来源：[akshare]，选择理由：[M2同比增速是衡量宏观流动性的核心指标，而流动性是影响股市走势的最直接因素之一。]
- 财务数据：[净资产收益率（ROE）、资产负债率]

### 存储方式
- 基础：CSV（方式 A）CSV 的优点在于极简通用，纯文本格式任何工具都能打开，体积小、传输快，且支持流式处理，能轻松应对超大文件。但它力不从心的情况也很明显：无法表达嵌套层级（如 JSON 的对象）、没有数据类型定义（数字和字符串看起来一样）、不支持多张表关联，而且处理文本中的逗号、换行符、引号时需要繁琐的转义，容易出错，不适合复杂数据结构或严格数据治理的场景。
- 进阶：Parquet（方式 B）
- 选择进阶方式的理由：Parquet为列式存储格式，拥有极高的压缩比。同样是存储 A 股全市场的历史日线数据，Parquet 文件占用的硬盘空间可能只有 CSV 或 SQLite 的 1/10 甚至更少。

4. 运行 `03_analysis.ipynb` 查看分析结果
5. 打开 `report.html` 阅读完整报告
